# Resonance Tester

Adds a `MEASURE_AXES_NOISE`, `TEST_RESONANCES`, and `SHAPER_CALIBRATE` macro to the printer

Look in `custom/resonance-tester.cfg`

## Commands Added


### Measure Axis Noise:

Measures noise of all enabled accelerometer chips

### Test Resonances:

Runs the resonance test for a specifed axis

### Shaper Calibrate:

Similar to `TEST_RESONANCES` but suggests input shaper config
